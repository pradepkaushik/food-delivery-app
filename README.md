🚀 DevOps Project #2 – Food Delivery App CI/CD Pipeline with Jenkins, Docker, Amazon ECR & Kubernetes (EKS)

I recently completed another hands-on DevOps project to strengthen my understanding of CI/CD and container orchestration.

In this project, I built a complete deployment workflow for a simple Food Delivery App, taking it from source code to a running application on Amazon EKS.

🔧 Tech Stack
Git & GitHub
Jenkins
Docker
Amazon ECR
Kubernetes (Amazon EKS)
AWS EC2
HTML
🚀 CI/CD Workflow

✅ Source code stored in GitHub
✅ Jenkins automatically pulls the latest code
✅ Docker image is built from the application
✅ Image is pushed to Amazon ECR
✅ Kubernetes deployment uses the latest container image
✅ Application is exposed using a Kubernetes Service

📁 Repository Structure
Dockerfile
Jenkinsfile
Kubernetes Deployment & Service YAML
HTML Application
💡 What I Learned
Writing a Jenkins Pipeline for automated builds
Building and tagging Docker images
Authenticating Jenkins with AWS
Pushing images securely to Amazon ECR
Deploying applications on Amazon EKS
Managing Kubernetes Deployments and Services
Understanding the complete CI/CD lifecycle from code commit to deployment

Every project helps me understand how modern DevOps workflows are implemented in real environments. Looking forward to building more projects involving monitoring, GitOps, and Infrastructure as Code.
